# Conventional Commits

Reference: [docs/developer/standard/conventional-commits.md](../../../docs/developer/standard/conventional-commits.md)

## Key Points

- Format: `type(scope): description`
- Use imperative mood in subject line
- 50 chars max in subject, wrap body at 72 chars
- Types are project-specific, defined in `atomi_release.yaml` — refer to generated `CommitConventions.md`
